# Hub Tech ITSM

Sistema de registro e acompanhamento de chamados técnicos da Hub Tech, para os contratos de terceirização de suporte.

Documentação completa (spec, MoSCoW, design system, roadmap) em [`docs/`](docs/asbuilt.md).

## Stack

Next.js · Supabase (Postgres + Auth + RLS) · Resend · Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

Variáveis de ambiente necessárias em `.env.local` (ver `docs/memoria/projeto.md`).
